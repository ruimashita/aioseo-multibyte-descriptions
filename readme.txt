== Description ==

Aioseo Multibyte Descriptions is  wordpress plugin for multibyte language user , work well with [All in One SEO Pack](http://wordpress.org/extend/plugins/all-in-one-seo-pack) autogenerating META descriptions.

== Installation ==

1. Download and Unzip.
1. Upload the aioseo-multibyte-descriptions foldero to your WordPress plugins directory inside of wp-content.  
      (/wp-content/plugins/aioseo-multibyte-descriptions).
1. Activate the plugin.
1. Setting the plugin options.
